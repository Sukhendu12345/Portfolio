# 🚀 আপনার পোর্টফোলিও ওয়েবসাইট

একটি সম্পূর্ণ Colorful/Circuit-থিমের পোর্টফোলিও ওয়েবসাইট, যার সাথে আছে একটি **Admin Panel** — যেখান থেকে আপনি সাইটের A থেকে Z সব কিছু (নাম, ছবি, স্কিল, প্রজেক্ট, কালার থিম, সব) এডিট করতে পারবেন। কোনো কোড লেখার দরকার নেই।

## 📁 ফাইল স্ট্রাকচার

```
portfolio/
├── index.html          ← মূল ওয়েবসাইট (ভিজিটররা এটা দেখবে)
├── admin.html          ← অ্যাডমিন প্যানেল (এখান থেকে এডিট করবেন)
├── css/
│   ├── style.css       ← সাইটের ডিজাইন/থিম
│   └── admin.css        ← অ্যাডমিন প্যানেলের ডিজাইন
└── js/
    ├── data.js          ← ডিফল্ট কন্টেন্ট + ডেটা সেভ/লোড ফাংশন
    ├── main.js          ← সাইট রেন্ডার করার লজিক
    └── admin.js         ← অ্যাডমিন প্যানেলের লজিক
```

সব ফাইল একসাথে রাখবেন — ফোল্ডার স্ট্রাকচার পরিবর্তন করবেন না, তাহলে লিংক ভেঙে যাবে।

---

## ⚠️ প্রথমেই এটা পড়ুন: লোকাল সার্ভার লাগবে

এই সাইট আপনার এডিট করা ডেটা **ব্রাউজারের localStorage**-এ সেভ করে। কিন্তু `index.html` ফাইলে শুধু ডাবল-ক্লিক করে (file:// দিয়ে) খুললে অনেক ব্রাউজারে localStorage কাজ করে না।

তাই টেস্ট করার সময় একটি ছোট লোকাল সার্ভার চালান:

**যদি আপনার কম্পিউটারে Python থাকে:**
```
cd portfolio
python -m http.server 8000
```
তারপর ব্রাউজারে যান: `http://localhost:8000/index.html` এবং `http://localhost:8000/admin.html`

**VS Code ইউজার হলে:** "Live Server" এক্সটেনশন ইনস্টল করে `index.html`-এর উপর রাইট-ক্লিক করে "Open with Live Server"।

GitHub Pages বা অন্য কোনো হোস্টিং-এ আপলোড করলে এই সমস্যা থাকবে না — সরাসরি কাজ করবে।

## 🔑 অ্যাডমিন প্যানেলে লগইন

1. `admin.html` খুলুন
2. ডিফল্ট পাসওয়ার্ড: **admin123**
3. লগইন করার পর "🔒 পাসওয়ার্ড" ট্যাব থেকে নিজের পাসওয়ার্ড সেট করুন (এটা শুধু আপনার ব্রাউজারে সেভ থাকে)

> ⚠️ এটি একটি ব্রাউজার-সাইড পাসওয়ার্ড — হাই-সিকিউরিটি দরকার এমন তথ্য (ব্যাংক/পাসওয়ার্ড) এখানে রাখবেন না। এটা শুধু সাধারণ ভিজিটরদের থেকে অ্যাডমিন প্যানেল আলাদা রাখার জন্য।

## ✏️ কীভাবে এডিট করবেন

অ্যাডমিন প্যানেলের সাইডবারে ১০টি ট্যাব আছে:

| ট্যাব | কী এডিট হবে |
|---|---|
| ⚙ সাইট ও থিম | সাইটের টাইটেল, লোগো টেক্সট, ৫টি থিম কালার, ফুটার |
| 🚀 হিরো সেকশন | নাম, রোল, ট্যাগলাইন, বাটন, প্রোফাইল ছবি, সোশ্যাল লিংক, রিজিউমি লিংক |
| 👤 আমার পরিচিতি | About বায়ো, ছবি, স্ট্যাটস (যেমন "10+ Projects") |
| 🛠 স্কিল | স্কিল ক্যাটাগরি ও প্রতিটি স্কিলের পার্সেন্টেজ |
| 📁 প্রজেক্ট | প্রজেক্টের নাম, বর্ণনা, ছবি, ট্যাগ, ডেমো/কোড লিংক |
| 🎓 অভিজ্ঞতা ও শিক্ষা | কাজের অভিজ্ঞতা ও শিক্ষাগত তথ্য (টাইমলাইন) |
| 💼 সার্ভিস | আপনি কী কী কাজ করেন (ফ্রিল্যান্স সার্ভিস ইত্যাদি) |
| ✉ যোগাযোগ | ইমেইল, ফোন, লোকেশন, সোশ্যাল লিংক |
| 📦 ডেটা | Export / Import / Reset |
| 🔒 পাসওয়ার্ড | অ্যাডমিন পাসওয়ার্ড পরিবর্তন |

কোনো লিস্ট-টাইপ সেকশনে (স্কিল, প্রজেক্ট, এক্সপেরিয়েন্স ইত্যাদি) **"+ যুক্ত করুন"** বাটনে ক্লিক করে নতুন আইটেম যুক্ত করতে পারবেন, এবং **"✕ মুছুন"** দিয়ে ডিলিট করতে পারবেন।

এডিট শেষে উপরের **"সেভ করুন"** বাটনে ক্লিক করুন — তাহলেই পরিবর্তন সাইটে প্রতিফলিত হবে। সেভ না করলে পরিবর্তন থাকবে না।

### ছবি যুক্ত করা
প্রতিটি ছবি ফিল্ডে দুটি অপশন:
- **URL দিন** — কোনো ছবি অনলাইনে (যেমন imgur.com, GitHub) আপলোড করে তার লিংক দিন (recommended)
- **ফাইল আপলোড করুন** — সরাসরি আপনার ডিভাইস থেকে ছবি বেছে নিন (ছোট সাইজ, ৫০০KB-র কম রাখার চেষ্টা করুন, কারণ localStorage-এর সাইজ লিমিটেড ~৫MB)

## 🎨 থিম কালার পরিবর্তন

"⚙ সাইট ও থিম" ট্যাবে ৫টি কালার আছে:
- **Primary** – মূল accent কালার (বাটন, হাইলাইট)
- **Cyan** – সেকশন নোড, লিংক হোভার
- **Pink** – সেকেন্ডারি accent
- **Amber** – এক্সপেরিয়েন্স সেকশনের হাইলাইট
- **Background** – সাইটের বেস ব্যাকগ্রাউন্ড কালার

কালার পরিবর্তন করে সেভ করলেই পুরো সাইটের লুক বদলে যাবে।

## 🌐 ফ্রি হোস্টিং (GitHub Pages)

1. GitHub-এ একটি নতুন রিপোজিটরি তৈরি করুন (যেমন `my-portfolio`)
2. এই পুরো `portfolio` ফোল্ডারের সব ফাইল ও সাবফোল্ডার আপলোড করুন
3. রিপোর **Settings → Pages**-এ যান
4. **Source: Deploy from a branch** সিলেক্ট করে `main` ব্রাঞ্চ ও `/ (root)` ফোল্ডার বেছে নিন → Save
5. কিছুক্ষণের মধ্যে আপনার সাইট লাইভ হবে: `https://<username>.github.io/my-portfolio/`
6. অ্যাডমিন প্যানেল: `https://<username>.github.io/my-portfolio/admin.html`

### হোস্ট করার আগে ডেটা সিঙ্ক করা
লোকালে এডিট করা ডেটা শুধু আপনার ব্রাউজারে আছে। হোস্ট করার আগে:
1. লোকাল অ্যাডমিন প্যানেল → "📦 ডেটা" ট্যাব → **"JSON এক্সপোর্ট করুন"** (একটি `portfolio-data.json` ফাইল ডাউনলোড হবে)
2. হোস্ট হয়ে যাওয়ার পর, হোস্টেড সাইটের অ্যাডমিন প্যানেল → "📦 ডেটা" ট্যাব → **"JSON ইম্পোর্ট করুন"** → এই ফাইলটি বেছে নিন → **"সেভ করুন"**

এভাবে দুই জায়গাতেই একই কন্টেন্ট থাকবে।

## 🛠 আরও কাস্টমাইজেশন (কোড লেভেলে)

যদি অ্যাডমিন প্যানেলের বাইরে আরও কিছু পরিবর্তন করতে চান:
- নতুন সেকশন/লেআউট: `index.html` + `css/style.css`
- রং/ফন্ট/স্পেসিং-এর বেস ভ্যারিয়েবল: `css/style.css`-এর `:root{}` অংশ
- কোনো নতুন ডেটা ফিল্ড যুক্ত করতে চাইলে: `js/data.js`-এ `DEFAULT_DATA` অবজেক্টে ফিল্ড যুক্ত করুন, `js/main.js`-এ render ফাংশনে দেখান, এবং `js/admin.js`-এ একটি ফিল্ড যুক্ত করুন
